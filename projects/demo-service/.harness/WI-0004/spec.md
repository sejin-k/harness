# 개요

`app` 패키지에 섭씨(Celsius) 온도를 화씨(Fahrenheit)로 변환하는 함수를 구현한다. 변환 공식은 `F = C * 9/5 + 32` 이며, 다양한 모듈에서 재사용할 수 있는 단일 진입점을 제공하기 위함이다.

## 범위

**다룰 것**
- `app` 패키지 내에 섭씨→화씨 변환 함수 1개 구현
- `int` 또는 `float` 입력을 받아 `float`를 반환
- 숫자가 아닌 입력에 대한 `TypeError` 발생 처리
- 위 동작을 검증하는 단위 테스트

**다루지 않을 것**
- 화씨→섭씨 등 역방향/기타 단위 변환
- CLI, HTTP API, UI 등 외부 인터페이스
- 국제화/반올림 정책, 유효 자릿수 제어
- 성능 최적화 및 동시성 처리

## 수용 기준 (Acceptance Criteria)

- [ ] `app` 패키지에서 변환 함수를 import 할 수 있다.
- [ ] `0`(섭씨)을 입력하면 `32.0`을 반환한다.
- [ ] `100`(섭씨)을 입력하면 `212.0`을 반환한다.
- [ ] `-40`(섭씨)을 입력하면 `-40.0`을 반환한다.
- [ ] `37.5`(float)를 입력하면 `99.5`를 반환한다.
- [ ] 입력이 `int`든 `float`든 반환 타입은 항상 `float`이다.
- [ ] 문자열(`"10"`) 입력 시 `TypeError`가 발생한다.
- [ ] `None` 입력 시 `TypeError`가 발생한다.
- [ ] `bool`을 숫자로 취급하지 않으며 `True`/`False` 입력 시 `TypeError`가 발생한다.

## 영향 받는 파일/모듈

- `app/` 패키지 — 변환 함수가 정의될 모듈(예: `app/temperature.py`) 신규 또는 수정
- `app/__init__.py` — 함수 공개(export)가 필요한 경우 수정
- `tests/` — 변환 함수 동작 및 예외를 검증하는 테스트 파일 신규 추가
