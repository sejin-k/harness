# 개요

문자열을 입력받아 단어 수를 세는 함수를 `app` 패키지에 제공한다. 공백 기준으로 단어를 분리하는 공통 로직을 한 곳에 모아 재사용성과 테스트 가능성을 확보하기 위함이다. 외부 의존 없이 명확한 규칙으로 단어 개수를 정수로 반환한다.

## 범위

**이번 작업에서 다룰 것**
- `app` 패키지에 단어 수 계산 함수 1개 제공
- 공백(스페이스 `' '`, 탭 `'\t'`, 개행 `'\n'` 등 모든 공백 문자) 기준 단어 분리
- 앞뒤 공백 및 단어 사이의 연속 공백 무시
- 빈 문자열 또는 공백만 있는 문자열에 대해 `0` 반환
- 입력이 문자열이 아닐 경우 `TypeError` 발생

**이번 작업에서 다루지 않을 것**
- 언어별 형태소/어절 분석, 유니코드 단어 경계(UAX #29) 정밀 처리
- 구두점 기반 분리(예: `"a,b"`를 2단어로 세는 것), 하이픈/언더스코어 처리
- 줄 수·문자 수 계산 등 단어 수 외의 통계
- 영속화(DB 저장), API 엔드포인트, UI

## 함수 시그니처(제안)

```python
# app 패키지 내
def count_words(value: str) -> int:
    ...
```

- 입력: `value` — 단어 수를 셀 대상 문자열
- 출력: 공백으로 분리된 단어의 개수(0 이상의 정수)
- 예외: `value`가 `str`이 아니면 `TypeError`

## 계산 규칙

1. `value`가 `str` 타입이 아니면 `TypeError`를 발생시킨다.
2. 모든 종류의 공백 문자(스페이스, 탭, 개행 등)를 단어 구분자로 사용한다.
3. 문자열 앞뒤의 공백은 무시한다.
4. 단어 사이의 연속된 공백은 하나의 구분자로 취급한다(빈 단어를 만들지 않는다).
5. 위 규칙 적용 후 남은 공백이 아닌 토큰의 개수를 반환한다.
6. 빈 문자열 또는 공백만으로 이루어진 문자열은 `0`을 반환한다.

## 수용 기준 / Acceptance Criteria

- [ ] `app` 패키지에서 단어 수 계산 함수를 import하여 호출할 수 있다.
- [ ] `"hello world"`를 입력하면 `2`를 반환한다.
- [ ] 단어가 하나뿐인 `"hello"`를 입력하면 `1`을 반환한다.
- [ ] 앞뒤 공백이 있는 `"  hello world  "`를 입력하면 `2`를 반환한다.
- [ ] 단어 사이 연속 공백이 있는 `"hello    world"`를 입력하면 `2`를 반환한다.
- [ ] 탭과 개행이 섞인 `"hello\tworld\nfoo"`를 입력하면 `3`을 반환한다.
- [ ] 빈 문자열 `""`을 입력하면 `0`을 반환한다.
- [ ] 공백만 있는 `"   "` 또는 `"\t\n "`을 입력하면 `0`을 반환한다.
- [ ] 문자열이 아닌 입력(예: `123`, `None`, `["hello"]`)을 전달하면 `TypeError`가 발생한다.

## 영향 받는 파일/모듈

- `app/` 패키지: 단어 수 계산 함수가 추가될 모듈(예: 기존 `app/validators.py` 또는 신설 모듈에 추가 후 `app/__init__.py`에서 노출)
- 테스트: 위 수용 기준을 검증하는 단위 테스트(예: `tests/test_count_words.py`)
