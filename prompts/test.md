# 할 일
명세의 **수용 기준 각 항목을 검증하는 자동화 테스트**를 작성하고, 전부 통과시켜라.

1. 레포의 기존 테스트 구조/관례를 파악해 동일한 방식으로 추가한다.
2. 테스트 실행 명령은 다음과 같다 (반드시 이 명령이 통과해야 한다):

       {test_cmd}

3. 테스트를 작성한 뒤 위 명령을 직접 실행해 통과를 확인하라.
4. 실패하면 구현 또는 테스트를 수정해 통과시켜라. (단, 명세 범위 내에서만)

# 제약
- 테스트를 무력화하거나(assert 제거 등) 형식만 통과시키는 행위 금지.
- 수용 기준을 실제로 검증해야 한다.
- 완료되면 추가한 테스트와 최종 실행 결과를 요약하라.
